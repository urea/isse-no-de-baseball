# 対戦キャラクター設定

## 方針

- セリフは飾りではなく、相手が選びやすいカードを読むためのヒントにする。
- ヒントは「そのカードを必ず選ぶ」という宣言ではない。各キャラクターには傾向とブラフがある。
- 難易度は反射神経ではなく、性格とセリフを読めるかで変える。
- 8体は対戦相手選択画面から選べる。セリフと基本的な選択傾向をゲームへ反映済み。

| キャラクター | 読み合いの軸 | ブラフ | 対戦感 |
|---|---|---:|---|
| Codex | 状況を分析して最も価値の高いカードを狙う | 少 | 素直な基準相手 |
| Dewey | 単打を選び、着実に進める | 少 | 初心者向け |
| Fireball | 長打を強気に狙う | 中 | 分かりやすいパワー型 |
| Hoots | プレイヤーの前回の選択を観察して外す | 高 | 読み返し型 |
| Rocky | 同じ方針を崩さず二塁打を好む | 少 | 安定型 |
| Seedy | 毎回違う発想で選び、HRも恐れない | 中 | 予測不能型 |
| Stacky | 走者・点差を見て期待値を積み上げる | 中 | 上級者向け |
| BSOD | 選択が乱れやすく、失敗も多い | なし | コメディ型 |
| Null Signal | ほぼ話さず、規則性を隠す | 高 | 最終盤向け |

## Codex

冷静に盤面を読む基準キャラクター。走者がいる時は長打、いない時は安定打も選ぶ。

- 選択前: 「盤面を整理しよう。」 / 「一番通る手を選ぶ。」
- ヒット時: 「想定どおりだ。」 / 「この一打で進めよう。」
- アウト時: 「その読みか。次を組み直す。」

## Dewey

きれいに一歩ずつ進める。単打を好み、無理な長打をあまり選ばない。

- 選択前: 「まずは、ひとつずつ。」 / 「急がなくても大丈夫。」
- ヒット時: 「きれいにつながったね。」
- アウト時: 「整えて、もう一度。」

## Fireball

速さと勢いを好むパワー型。二塁打以上が見えていると、隠し切れないことが多い。

- 選択前: 「長打だけは打たせん！」 / 「一発、狙っていくぜ！」 / 「派手に決めようか！」
- ヒット時: 「もらったァ！」 / 「火がついたぜ！」
- アウト時: 「チッ、次は逃がさない！」

## Hoots

観察眼の鋭い読み返し型。プレイヤーが前回選んだ位置や、わかりやすい長打カードを逆手に取る。

- 選択前: 「その視線、見えているよ。」 / 「前と同じ手かな？」
- ヒット時: 「見通しどおり。」
- アウト時: 「ほう、そこを選んだか。」

## Rocky

大きな差分にも動じない安定型。二塁打を中心に、毎回似た基準で選ぶ。

- 選択前: 「焦らず、確かな一手を。」 / 「崩さず進もう。」
- ヒット時: 「積み上げた結果だ。」
- アウト時: 「土台は揺らがない。」

## Seedy

新しい芽を探すひらめき型。定石よりも、普段なら選ばれないカードへ飛びつく。

- 選択前: 「こっちに、なにかありそう！」 / 「新しい一手、試してみよう。」
- ヒット時: 「ほら、芽が出た！」
- アウト時: 「今回は土の中だったかも。」

## Stacky

深く積み上げる計算型。走者がいる時は得点期待値、いない時は次の布石を優先する。

- 選択前: 「順番に積めば、答えになる。」 / 「ここは期待値を取る。」
- ヒット時: 「一段ずつ、上へ。」
- アウト時: 「スタックを積み直す。」

## BSOD

調子が乱れやすいコメディ型。セリフと選択が一致しないのはブラフではなく、本人も混乱しているため。

- 選択前: 「えっと……こっち？」 / 「一瞬、画面が青くなった。」
- ヒット時: 「たまたま通った！？」
- アウト時: 「例外が発生しました……。」

## Null Signal

静かな最終盤向け。言葉が少なく、選択傾向も見えにくい。

- 選択前: 「……」 / 「signal lost」
- ヒット時: 「……confirmed」
- アウト時: 「……noise」
